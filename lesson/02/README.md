## ファイルの変更を監視して自動でtaskを実行する

`gulp.watch`を利用することで、ソースファイルが変更される度に
自動でタスクを実行することが可能になります。

### 準備

1. watchタスクを作成する
1. 実行

    ```js
    $ gulp watch
    ```
